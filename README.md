🌦️ Weather App

A simple and interactive Weather App built using HTML, CSS, and JavaScript that provides real-time weather information for any city.

🚀 Features

Search weather by city name

Displays temperature, humidity, and weather condition

Real-time data using API

Clean and responsive UI

🛠️ Technologies Used

HTML

CSS

JavaScript

API

Weather API (e.g., OpenWeather API)

▶️ How to Use

Open the app in your browser

Enter a city name

Click the search button

View current weather details

📌 Description

This project helps beginners understand how to work with APIs, fetch data, and update the UI dynamically using JavaScript.
